# spam-detection-


📌 Overview

This project builds a machine learning model to classify emails as spam or ham (not spam) using Natural Language Processing (NLP). It includes text preprocessing, feature extraction, model training, and evaluation.

⚙️ Features

Text cleaning (remove HTML, punctuation, special characters)

Stopword removal and stemming

Feature extraction using CountVectorizer and TF-IDF

Training multiple machine learning models

Performance evaluation using key metrics

Model saving using Pickle

🧠 Models Used: Logistic Regression,Multinomial Naive Bayes,Support Vector Machine (SVM) ,Random Forest,Gradient Boosting ,AdaBoost,Decision Tree,XGBoost.

📊 Results

Accuracy up to 97%

Precision up to 98%

Recall up to 94%
